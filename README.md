## 运行

git clone 代码到本地，然后启动

```
npm install
npm run start
```

PS：项目用到了 [tailwindcss](https://tailwindcss.com/) 开发样式，这对 React 学习没有影响

## 功能
* 添加 Todo 默认添加时间
* 浏览器本地存储
